# 🎮 Gamify – Cognitive Training Web App

A modern and interactive browser-based cognitive training application built using **HTML, CSS, and JavaScript**. Gamify combines multiple brain-training mini-games into a single platform to enhance reaction speed, memory, logic, and focus.

🌐 **Live Demo:** https://gamify-brain-games.netlify.app/

---

## 📌 Project Description

Gamify is a single-page web application designed to improve cognitive abilities through engaging gameplay. It includes multiple mini-games such as reaction tests, memory challenges, typing speed tests, and an arcade-style asteroid game.

The project focuses on **clean UI/UX**, **real-time interactivity**, and **efficient game logic implementation**, all within a lightweight, single-file architecture.

## 🏗️ System Architecture

### 1. Frontend Design
- Built using HTML, CSS for layout and styling  
- Responsive UI with modern design principles  
- Reusable components and structured layout  

### 2. Game Logic
- Implemented using Vanilla JavaScript  
- Independent logic for each mini-game  
- Event-driven interactions and state management  

### 3. Data Handling
- LocalStorage used for saving leaderboard data  
- Real-time updates for scores, streaks, and stats
